## Autentisering

Alla anrop kräver två http headers för att autentisera anropet. Content-Type bör också defineras för bästa upplevelse.

```
X-Portal-Id: '{portalId}'
Authorization: 'Bearer {token}'
Content-Type: 'application/json'
```

1. X-Portal-Id
   1. ID för portal som anropet ska hantera. Alla ID som finns tillgängliga för använder går att hämta genom anrop till /me
1. Authorization
   1. Standard http header för autentisering. Token kan skapas för en användare under inställningar i gränssnittet (https://app.eldra.ai/settings)

### Kom igång

<Accordion title="Autentisering" icon="user">
  Skapa en token för din användare under inställningar: https://app.eldra.ai/settings 

  Där kan du även se alla portal-ID som din användare har tillgång till. 

  Skicka sedan med följade headers i alla anrop:

  ```
  X-Portal-Id: '{portalId}'
  Authorization: 'Bearer {token}'
  Content-Type: 'application/json'
  ```
</Accordion>
